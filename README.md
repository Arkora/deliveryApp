# DeliveryApp

## Delivery App is a MERN Stack project was created for learning purposes.

This project is simple delivery app with real time ordering and getting orders from clients.
Admin can create,update,delete and read from database simply CRUD

## Features

- Simple UI 

- Real Time Data Rendering

- Can add one or more food in the cart section 

- Automatically total bill will generate.

- Search different food/types

- Select Ingrendients for each food
